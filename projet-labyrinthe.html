Jeu de Labyrinthe

Projet réalisé en 2025 - BUT Informatique Semestre 3

← Retour aux réalisations

Informations du projet

  • Année : 2025
  • Contexte : BUT Informatique - Semestre 3
  • Équipe : 4 personnes
  • Durée : 2 mois

Description du projet

Dans le cadre de notre formation en BUT Informatique, nous avons développé un jeu de labyrinthe complet en équipe de 4 personnes. Ce projet nous a permis de mettre en pratique nos compétences en programmation et en travail collaboratif.

Le jeu propose une expérience immersive avec des labyrinthes générés de manière procédurale, garantissant une rejouabilité optimale. Le joueur doit trouver la sortie du labyrinthe tout en relevant différents défis selon le mode de jeu choisi.

Fonctionnalités principales

  • Génération aléatoire des labyrinthes : Chaque partie propose un labyrinthe unique généré par un algorithme de génération procédurale, assurant une expérience différente à chaque session.
  • Plusieurs niveaux de difficulté : Du niveau débutant au niveau expert, le joueur peut choisir la complexité du labyrinthe (taille, nombre d'obstacles, temps imparti).
  • Différents modes de jeu : Mode classique, mode contre-la-montre, mode exploration... Plusieurs façons de jouer pour satisfaire tous les types de joueurs.

Technologies utilisées

Java JavaFX Algorithmes de génération Git Travail en équipe

Compétences développées

  • Programmation orientée objet avancée
  • Implémentation d'algorithmes de génération procédurale
  • Conception d'interfaces graphiques avec JavaFX
  • Gestion de projet en équipe
  • Versioning avec Git et GitHub
  • Tests et débogage
